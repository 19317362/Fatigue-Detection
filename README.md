# Omoshiroi
[![Build Status](https://travis-ci.org/Martin20150405/Omoshiroi.svg?branch=master)](https://travis-ci.org/Martin20150405/Omoshiroi) [![license](https://img.shields.io/github/license/mashape/apistatus.svg)](LECENSE)  ![progress](http://progressed.io/bar/15?title=Progress)

Omoshiroi aims being a fully-featured Camera/PhotoEditor/VideoRecorder app for Android, with hundreds of realtime filters (including GPU-image like filter-chain & visual effects & Face Beautify & Face Swap)  like Snapchat / FaceU /SNOW Cam /VSCO -安卓实时滤镜、美颜相机、换脸贴纸、拍照录像录屏 
 
* Up to 2017.04.07, there are 107 different filters.
